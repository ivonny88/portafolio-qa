Sobre mi

Soy QA Junior con una fuerte motivación por asegurar la calidad del software y mejorar continuamente los procesos de desarrollo. Me caracterizo por mi atención al detalle, la constancia y una mentalidad analítica orientada a detectar incidencias y proponer soluciones efectivas. Además de mi experiencia en aseguramiento de calidad, cuento con conocimientos de SEO, adquiridos gracias a la creación y gestión de varias webs propias monetizadas con AdSense. Esto me aporta una visión más amplia del producto digital, el comportamiento del usuario y la optimización orientada a resultados. Disfruto especialmente del trabajo en equipo, la comunicación clara y el aprendizaje constante. Busco una oportunidad en la que pueda seguir creciendo como QA, aportar valor desde el primer día y formar parte de un entorno colaborativo y dinámico.

Proyectos

Portfolio QA Web – QA Manual, Automatización y Desarrollo Web

Desarrollé un portafolio web para mostrar proyectos de QA utilizando HTML5, CSS3, Visual Studio y control de versiones con Git/GitHub.

Diseñé y ejecuté pruebas de regresión manuales y automatizadas con Katalon Studio (basado en Selenium), verificando la funcionalidad del sitio.

Implementé un flujo de integración continua con GitHub Actions para ejecutar automáticamente las pruebas de regresión y generar reportes en cada cambio del repositorio.

API de Usuarios – QA API Testing

Diseñé una base de datos de usuarios y generé automáticamente una API REST utilizando SQLbase, a partir de la estructura de la tabla.

Creé una colección de pruebas en Postman con validaciones automáticas para casos positivos, negativos y autenticación, verificando códigos de respuesta y contenido esperado.

Verifiqué con SQL que los datos se almacenaban y actualizaban correctamente en la base de datos tras cada petición a la API.

Implementé un flujo de integración continua (CI) con GitHub Actions, utilizando Newman para ejecutar las pruebas automáticamente desde la colección de Postman y generar reportes diarios en el repositorio.
